## ✅ 第三部分：选择与使用指南
### 🧭 如何选择？
- **个人开发者/学生**
  - **首选免费**：百度ERNIE Speed/Lite、讯飞spark lite、智谱GLM-4-Flash。
  - **低价备用**：豆包Pro-32k、通义千问Qwen1.5-7B。
- **中小企业/创业公司**
  - **性价比首选**：豆包Pro-32k、DeepSeek-V3。
  - **中文场景**：通义千问Qwen1.5-7B、文心一言4.5 Turbo。
- **大型企业/高并发需求**
  - **企业级**：文心一言X1 Turbo、通义千问Qwen-Max。
  - **长文本处理**：智谱GLM-4(128K)、Kimi（月之暗面）。
### 💡 省钱技巧
1.  **利用缓存**：阿里云百炼、通义千问支持缓存，命中后费用大幅降低。
2.  **批量调用**：部分平台（如阿里云百炼）的batch调用可享半价。
3.  **聚合平台**：通过七牛云AI、SiliconCloud等平台统一管理，价格更低，访问更快。
4.  **开源模型**：通过Hugging Face或自部署开源模型，长期成本更低。
### 📌 调用示例
以下是一个调用智谱AI免费模型的Python示例：
```python
from zhipuai import ZhipuAI
client = ZhipuAI(api_key="YOUR_API_KEY")  # 请替换为你的API Key
response = client.chat.completions.create(
    model="glm-4-flash",  # 使用免费模型
    messages=[{"role": "user", "content": "你好，请介绍一下你自己。"}],
)
print(response.choices[0].message.content)
```
